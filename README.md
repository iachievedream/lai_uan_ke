# lai_uan-ke 來吵架

取「來吵架」的台語「lai uan-ke」

用酸民最常使用的留言板來做個人 Laravel 練習

## 當前功能
+ 單頁式網頁
+ 進入必須設定使用者暱稱，否則會看不到內容頁面(被導到設定頁面)
+ 使用者輸入留言，送出
+ 留言在下方條列式顯示，採日期降冪排列
+ 留言顯示作者，冤有頭債有主

## 未來功能
+ 使用者登入
+ 註冊(email, facebook)
+ 留言的下方回覆留言，讓吵架更順暢
